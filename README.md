# Demo
## Usage
### install
```shell
# for naive-ui
npm i -D naive-ui
npm i -D vfonts
# for xicons
npm i -D @vicons/fluent
npm i -D @vicons/ionicons4
npm i -D @vicons/ionicons5
npm i -D @vicons/antd
npm i -D @vicons/material
npm i -D @vicons/fa # font awesome
npm i -D @vicons/tabler
npm i -D @vicons/carbon
# for vue-router
npm i -D vue-router@next  
# for axios
npm i -D axios   
```
### run 
```shell
npm run serve
```
